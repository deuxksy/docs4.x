# 펌웨어 초기화

웹 관리 패널 왼쪽에서 **SYSTEM** -> **Reset Firmware**로 이동합니다.

라우터에 오동작이 발생하면 펌웨어를 초기화하여 문제를 해결할 수 있습니다. **Delete All and Reboot** 버튼을 클릭하여 펌웨어를 초기화하세요.

**참고:** 현재 모든 설정, 애플리케이션 및 데이터가 손실됩니다. 프로세스는 약 2분 정도 소요됩니다. 이 프로세스 중에는 라우터의 전원을 끄지 마세요.

![reset firmware](https://static.gl.inet.com/docs/router/en/4/tutorials/reset_firmware/reset_firmware_4.8.png){class="glboxshadow"}

웹 관리 패널에 액세스할 수 없는 경우 아래 동영상을 보고 리셋 버튼을 사용하여 초기화하세요.
