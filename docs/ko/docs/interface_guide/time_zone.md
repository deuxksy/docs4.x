# 시간대

웹 관리 패널 왼쪽에서 **SYSTEM** -> **Time Zone**으로 이동합니다.

라우터는 시스템 시간에 따라 활동 타임스탬프를 기록하며 일부 기능은 올바른 시간에 의존하여 제대로 작동합니다. 올바른 시간대가 선택되고 동기화되었는지 확인하세요.

시간대는 자동으로 동기화되지 않습니다. **Sync** 버튼을 클릭하여 업데이트하세요.
