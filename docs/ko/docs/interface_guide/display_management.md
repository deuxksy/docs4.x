# 디스플레이 관리

**참고**: 이 기능은 **GL-BE3600 (Slate 7)** 및 **GL-E5800 (Mudi 7)** 에서 사용할 수 있습니다.

웹 관리 패널 왼쪽의 **SYSTEM** -> **Display Management** 로 이동합니다.
이 페이지에서는 터치스크린 디스플레이와 관련 설정을 관리할 수 있습니다.

## GL-BE3600

![display management](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_display.png){class="glboxshadow"}

- **Display Function**: 터치스크린에 표시할 기능을 관리합니다.

    ![function](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_function.png){class="glboxshadow"}

- **Lock Screen**: 잠금 화면 설정을 사용자 지정합니다. 여기에는 절전 디스플레이와 배경화면이 포함됩니다.

    ![lock screen](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_lockscreen.png){class="glboxshadow"}

    배경화면을 **Customize** 로 전환하면 최대 5MB 크기의 사용자 지정 이미지를 업로드할 수 있습니다.

    ![wallpaper customize](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_wallpaper_custom.png){class="glboxshadow"}

- **Brightness**: 터치스크린 밝기를 조정합니다. 슬라이더를 사용하거나 특정 레벨(1~10 범위)을 입력하여 다양한 조명 조건에 맞춥세요.
- **Auto Lock**: 활동이 없을 때 화면이 자동으로 잠기는 시간 지연을 설정합니다. 범위는 1~30분입니다.
- **Screen Always On**: 터치스크린을 항상 켜진 상태로 유지할지 비활동 후 꺼질지를 결정합니다.
- **Enable Screen Passcode**: 터치스크린에 비밀번호(4자리 숫자만)를 활성화하여 추가 보안 계층을 추가합니다.

    ![screen passcode](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_passcode.png){class="glboxshadow"}

## GL-E5800

![display management](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/e5800_display.png){class="glboxshadow"}

- **Wallpaper**: 배경화면과 절전 디스플레이 스타일을 사용자 지정합니다.
- **Brightness**: 터치스크린 밝기를 조정합니다. 슬라이더를 사용하거나 백분율을 입력하여 다양한 조명 조건에 맞춥세요.
- **Personalised Signature**: 터치스크린에 사용자 지정 텍스트를 추가하여 고유한 스타일을 표현하거나 빠른 식별을 위해 사용합니다.
- **Auto Lock**: 활동이 없을 때 화면이 자동으로 잠기는 시간 지연을 설정합니다. 범위는 15초~5분입니다.
- **Passcode**: 터치스크린에 비밀번호를 설정하여 추가 보안 계층을 추가합니다.
