# 플러그인

웹 관리 패널 왼쪽에서 **APPLICATIONS** -> **Plug-ins**로 이동합니다.

이 페이지에서 OpenWrt 패키지를 관리할 수 있습니다. 필요에 따라 패키지를 설치하거나 제거할 수 있습니다.

서드파티 플러그인을 사용하기 전에 **Update** 버튼을 클릭하는 것이 좋습니다. 하지만 `nginx`, `adguard home`, `tailscale` 등 **공식 GL.iNet 펌웨어의 미리 설치된 플러그인은 업데이트하지 마세요**.

![Plug-ins](https://static.gl.inet.com/docs/router/en/4/tutorials/plug-ins/plug-ins.png){class="glboxshadow"}

사용자가 수동으로 설치한 서드파티 플러그인은 업데이트할 수 있습니다. 단, GL.iNet은 서드파티 플러그인의 보안 문제에 대해 책임지지 않습니다. 문의는 해당 서드파티 플러그인 작성자에게 문의하세요.
