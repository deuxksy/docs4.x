# 콘텐츠 필터

콘텐츠 필터는 DPI 기반 분류를 통해 스마트한 온라인 안전을 제공하며, 유해하거나 악성 웹사이트를 자동으로 차단하여 네트워크를 깨끗하고 안전하게 유지합니다.

**참고**: 이 기능은 현재 **GL-MT5000 (Brume 3)** 에서만 사용할 수 있습니다.

---

웹 관리 패널 왼쪽의 **FLOW CONTROL** > **Content Filter** 로 이동합니다.

차단할 콘텐츠(앱, 도메인 또는 IP 주소)를 사용자 지정하고 **Apply** 를 클릭합니다.

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List**: 이 섹션에는 도박, 성인, 악성코드의 세 가지 미리 선택된 카테고리가 포함되어 있습니다. 활성화하면 이 세 가지 카테고리와 관련된 모든 웹사이트, 서비스 또는 앱이 차단됩니다. 필요에 따라 Edit App을 클릭하여 다른 카테고리(예: 게임, 소셜 미디어, 쇼핑)를 추가할 수 있습니다.

- **Blocked Domain / IP List**: 이 섹션에서는 특정 도메인(예: google.com) 또는 IP 주소(예: 192.168.8.0/24)를 수동으로 입력하여 해당 주소로의 접근을 차단할 수 있습니다. 목록은 최대 10000개 항목까지 지원합니다.
