# DPI 라이선스

DPI(Deep Packet Inspection)는 지능형 네트워크 관리의 핵심 기능입니다. 전통적인 라우터의 한계(출발지 또는 목적지 주소만 식별)를 극복하고, 데이터 패킷 페이로드를 심층 분석하며, 특징 라이브러리 비교를 통해 사용자가 액세스한 애플리케이션과 웹사이트를 정확하게 식별하여 정교한 트래픽 분류와 제어를 가능하게 합니다.

[Netify](https://www.netify.ai/){target="_blank"}와 통합된 GL.iNet DPI는 효율적인 배포를 위해 경량 임베디드 플러그인을 채택했습니다. Netify의 온라인 업데이트 시그니처 데이터베이스를 통해 안정적인 관리를 제공하여 네트워크 제어를 더욱 정확하고 효율적으로 만듭니다.

**참고**:

1. DPI 기능은 현재 **GL-MT5000 (Brume 3)** 에서만 사용할 수 있습니다.

2. 라우터가 드롭인 게이트웨이 모드인 경우 DPI 기능이 적용되지 않습니다.

---

라우터의 웹 관리 패널에 로그인하고 **FLOW CONTROL** > **DPI License** 로 이동합니다.

**30일 무료 체험** 을 시작하여 **데이터 통계**, **콘텐츠 필터**, **QoS** 를 포함한 DPI 기능을 사용하세요.
