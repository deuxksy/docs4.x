# IGMP Snooping

웹 관리 패널 왼쪽에서 **NETWORK** -> **IGMP Snooping**로 이동합니다.

IGMP snooping을 활성화하여 라우터에서 멀티캐스트 기능을 사용할 수 있습니다.

IGMP Snooping은 IGMP 프로토콜 패키지를 수신하고 해당 정보를 추출하여 2계층 멀티캐스트 포워딩 테이블을 설정하고 유지한 다음, 멀티캐스트 그룹 데이터를 멀티캐스트 그룹에 가입한 호스트로 포워딩하며 다른 호스트는 멀티캐스트 그룹 데이터를 수신할 수 없습니다.

**참고**: IGMPv3는 v1 및 v2와 호환됩니다. 기본적으로 v3를 사용하며 문제가 발생하면 전환하세요.
