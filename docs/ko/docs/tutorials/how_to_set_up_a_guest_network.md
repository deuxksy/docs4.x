# GL.iNet 라우터에 게스트 Wi-Fi 네트워크 설정하기

이 튜토리얼에서는 GL.iNet 라우터에 게스트 Wi-Fi 네트워크를 설정하는 방법을 안내합니다.

게스트 Wi-Fi 네트워크는 기본 장치에서 격리하려는 장치를 위한 라우터의 별도 네트워크입니다. 여기에는 게스트 장치와 스마트 홈 장치가 포함됩니다. 게스트 네트워크를 만들면 기본 네트워크에 대한 무단 액세스와 잠재적 공격을 방지하여 네트워크 보안을 강화할 수 있습니다.

GL.iNet 라우터에 게스트 Wi-Fi 네트워크를 설정하는 방법은 두 가지가 있습니다: [라우터 관리 패널](#관리-패널을-통한-게스트-네트워크-설정) 또는 [GL.iNet 모바일 앱](#glinet-모바일-앱을-통한-게스트-네트워크-설정)을 통하는 방법입니다.

## 관리 패널을 통한 게스트 네트워크 설정

### 1. 게스트 Wi-Fi 네트워크 활성화

1. 웹 브라우저에서 `192.168.8.1`을 입력합니다.
2. 비밀번호를 입력한 후 **Login**을 클릭합니다.
3. 왼쪽 사이드바에서 **Wireless**를 클릭합니다.
![무선 클릭](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/click-wireless.jpeg){class="glboxshadow"}
4. 5GHz 또는 2.4GHz Wi-Fi 네트워크 섹션에서 **Guest Wi-Fi** 탭을 클릭합니다.
![게스트 클릭](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/click-guest-wifi.jpeg){class="glboxshadow"}
5. **Enable Wi-Fi** 옆에서 스위치를 켭니다.

### 2. 새 게스트 Wi-Fi 네트워크 비밀번호 설정

보안을 위해 게스트 Wi-Fi 네트워크에 새 비밀번호를 설정하는 것이 좋습니다. 다음 단계를 따르세요:

1. **Modify**를 클릭합니다.
![수정 클릭](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/click-modify.jpeg){class="glboxshadow"}
2. **Wi-Fi Password** 옆에 새 비밀번호를 입력합니다. (**Wi-Fi Name** 옆에 새 네트워크 이름도 입력할 수 있습니다.)
3. 그런 후 **Apply**를 클릭합니다.

이제 게스트 Wi-Fi 네트워크가 설정되었습니다. 장치를 연결하려면 사용 가능한 네트워크 목록에서 게스트 네트워크 이름을 찾아 비밀번호를 입력합니다.

## GL.iNet 모바일 앱을 통한 게스트 네트워크 설정

**참고:** 시작하기 전에 장치에 GL.iNet 모바일 앱을 설치하고 설정하세요.

### 1. 게스트 Wi-Fi 네트워크 활성화

1. 메인 앱 화면에서 **Network** 탭 > **Wireless**를 탭합니다.
2. **Guest Network**를 찾습니다.
3. 게스트 Wi-Fi 네트워크를 설정하려는 네트워크의 스위치를 켭니다.
![스위치 토글](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/toggle-on-guest-network.jpeg){class="glboxshadow"}

### 2. 새 게스트 Wi-Fi 네트워크 비밀번호 설정

보안을 위해 게스트 Wi-Fi 네트워크에 새 비밀번호를 설정하는 것이 좋습니다. 다음 단계를 따르세요:

1. 게스트 Wi-Fi 네트워크를 설정한 네트워크의 **SSID** 옵션을 탭합니다.
2. **Password** 옆에 새 비밀번호를 입력합니다. (SSID 필드에 새 네트워크 이름도 입력할 수 있습니다.)
![새 비밀번호 입력](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/tap-password.jpeg){class="glboxshadow"}
3. **Done**을 탭합니다.

이제 게스트 Wi-Fi 네트워크가 설정되었습니다. 장치를 연결하려면 사용 가능한 네트워크 목록에서 게스트 네트워크 이름을 찾아 비밀번호를 입력합니다.
