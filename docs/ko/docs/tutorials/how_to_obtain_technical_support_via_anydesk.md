# AnyDesk를 통해 기술 지원 받기

**원격 지원을 통해 기술자가 네트워크 설정을 직접 확인하고 네트워크를 빠르게 구성하거나 네트워크 문제를 해결하는 데 도움을 줄 수 있습니다.**

**원격 지원을 수락하기 전 다음과 같이 몇 분 동안 준비해 주세요:**

## 1. 불필요한 애플리케이션 닫기

필요하지 않은 애플리케이션을 미리 모두 닫아주세요. 이는 주의 산만을 줄이고 개인 정보를 보호하는 데 도움이 됩니다.

## 2. 네트워크 안정성 확인

컴퓨터에서 Wi-Fi와 유선 연결을 모두 사용하여 원격 지원 세션 중 연결 끊김이나 지연을 방지하는 것이 좋습니다.

## 3. AnyDesk 다운로드 및 설치

공식 [AnyDesk 웹사이트](https://anydesk.com/)를 방문하여 운영 체제에 적합한 버전을 다운로드하세요.

![download anydesk](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/install_anydesk.png){class="glboxshadow"}

설치 후 AnyDesk를 실행합니다.

## 4. AnyDesk 주소 얻기

AnyDesk가 실행되면 숫자 주소(일반적으로 "123 456 789" 형식)가 표시됩니다. 이것이 지원 기술자와 공유해야 하는 주소입니다.

![get anydesk address](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/anydesk_address.png){class="glboxshadow"}

## 5. AnyDesk 주소 공유

지원 담당자에게 AnyDesk 주소를 보내 장치에 연결할 수 있도록 합니다.

## 6. 연결 요청 수락

기술자가 연결을 시도하면 연결 요청에 대한 프롬프트가 표시됩니다.

기술자를 신뢰하는지 확인한 후 "Accept"를 클릭하여 장치에 액세스할 수 있도록 합니다.

![accept request](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/accept_request.png){class="glboxshadow"}

## 7. 세션 중 커뮤니케이션

지원 세션 전체에서 스피커, AnyDesk 세션의 채팅 창 또는 기타 방법을 통해 자유롭게 커뮤니케이션하여 문제가 효과적으로 해결되도록 하세요.

## 8. 세션 종료

지원 세션이 완료되면 AnyDesk 인터페이스의 빨간 "End Session" 버튼을 클릭하여 연결을 종료합니다. 이렇게 하면 모든 원격 액세스 권한이 종료됩니다. 완전한 개인 정보 보호를 위해 다음도 수행하세요:

- AnyDesk 애플리케이션을 완전히 닫습니다 (최소화하지 않음).
- 잔여 프로세스에 우려가 있는 경우 컴퓨터를 다시 시작합니다.
- 시스템 설정을 확인하여 활성화된 원격 액세스 권한이 남아 있지 않은지 확인합니다.
